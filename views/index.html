Echo JSON

Usage

A request to http://echojson.com/hello/[1,2,3] will return the following JSON.

A request to http://echojson.com/hello/{"a":1} will return the following JSON.

You can also post JSON to any url and get the JSON you posted as returned.

Supports JSONP via ?callback= Turn off history with ?history=false
